Changes from 1.5.0 to 1.5.1
---------------------------

REMOVE	Removed custom database configuration scripting capability.
REMOVE	Removed interactive mode of the command line version of SolidBase.

CHANGE	Complete overhaul of the solidbase.properties format. It is now consistent with the capabilities of the Ant task and Maven plugin.
CHANGE	Maven plugin: renamed configuration item 'user' to 'username'.
CHANGE	Ant task: renamed configuration item 'user' to 'username'. But 'user' still works.

ENHANC	Added secondary connections capability to the Maven plugin.
ENHANC	Introduced new Ant task 'upgradedb' which is an alias of 'solidbase-upgrade'.
ENHANC	Improved exception handling.
ENHANC	Javadoc improvements.
